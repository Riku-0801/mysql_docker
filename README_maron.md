## 使い方

```
bash-4.4# mysql -u root -p
Enter password:
mysql> source docker-entrypoint-initdb.d/<filename>.sql
```

## 各 SQL ファイルの説明

- init.sql

  - 初期化用
  - 最初に実行する
  - 1 回だけ実行してください

- check.sql

  - 各テーブルの内容を全部出力する

- 4-n_m.sql
  - 4.n の小さい問 m の解答
